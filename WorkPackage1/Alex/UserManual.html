1 Three main views

The main window contains three panels: a top ribbon that stores functions related files, windows, and display; a left pane that gives you an overview of the rail, and a right pane that shows you detailed information and allows you to schedual and dispatch trains, and control switches.

1.1 Top ribbon

The top ribbon stores functions related to files, windows, and displays. From left to right, the drop down menues on the ribbon are:
File | Edit | Window | Help

1.1.1 File

The file drop down allows you to exit the CTC window

1.1.2 Edit

The edit drop down allows you to:

  • copy a section of text to your clipboard with the copy button,
  • to copy to your clipboard and delete the selected text with the cut button,
  • or to paste the text in the currently selected text edit area with the paste button.
1.1.3 Window

The window drop down allows you to open any of the information views in their own window, for more information on each information view, see section 1.3

1.1.4 Help

The help drop down contains two options,

  • About will list the authors and name of the software
  • Manual will open this page in your default browser

1.2 Left pane

The left pane shows a diagram of the track

1.2.1 Track Symbols

The track is depicted with a 2 pixel line, additionally, tracks with special designation use different symbols:

  • Track designated Switch will show it's activated rail normally, and any non-active rails in a faded color.
  • Track designated Station will have additional 4px lines on either side of the track, the same color as the track.
  • Track designated Crossing will have additional 2px blue lines on either side of the track.
1.2.2 Train Symbols

The trains are depicted in this pane as rectangles. Rectangles with an arrow inside are engines, the arrow points the same direction that the engine is faceing.

1.2.3 Colors

The 2px track can assume the following colors for the following reasons:

  • Normal(green)
  • Broken Rail(Gray)
  • Track Circuit Failure(Red)
  • Power Failure(Yellow)

If the section of track is experienceing more than one failure at a time, it will be striped with the colors of each failure in regular intervals

1.3 Right pane

The right pane shows more information and contains the controls for elemenets selected in the left pane.

1.3.1 Train schedule pane

The train schedule pane allows you to schedual setting the authority of a train at a future time. For dispatching a train immediately, see section 1.3.4

1.3.2 Train line visibility pane

The train line visibility pane allows you so set which lines are visible in the left pane. A checked box indicates that the line is currently visible. To toggle the visibility of a line, click on the checkbox next to it's label.

1.3.3 Train details pane

The train details pane displays details about the currently selected train. To select a train, simply click on it in the left pane. This pane will show you the train's name, throughput of the track it is currently on, current number of passengers, current speed, and current authority.

1.3.5 Message dialog pane,

The message dialog pane will display messages about the transit line, the messages may include rail failures, track switches, and trains executing their schedual. This is neither a complete or exhaustive list. More recent messages are displayed at the bottom, messages that have scrolled past the top of the pane can be viewed by scrolling up.
