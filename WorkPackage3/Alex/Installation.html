Installation Manual for CTC office

Running

The CTC comes with both the source files to build the ctc from scratch, as well as the compiled .class files nessessary to run it without compileing. To run the CTC without compileing, double click "run.bat". Keep in mind the CTC comes with a faux Track Controller module. To create a substitute track controller, see Compileing

Compiling

To substitue in a different Track Controller module, be sure that your track controller implements all the methods specified in the example Track Controller module located at /src/CTC/dummy/TrackController.java. Once you have your real track controller class, paste it over the example class and double click compile.bat, found in /src/CTC/compile.bat.

Modification of track

To modify the track file, open track.txt in your favorite text editor. Documentation on how the track is written is at the top. If, after changes and running the software you are getting an error within the range 101-199, see section 3.2.1 of the Users Manual

For referance, these are they way track segments are formated. Track formats:
Straight track
str, x, y, rotation, length, head, tail, headto(Depreciated), tailto(Depreciated), id, trackcontroler_id
ex.
str, 5, 10, 0, 100, 3, 7, TAIL, TAIL, 0, 0

Curved track
cur, x, y, start_angle, angle_length, radius, block_clockwise, block_counterclockwise, block_clockwise_to(Depreciated), block_counterclockwise_to(Depreciated), block_id, trackcontroller_id
ex.
cur, 20, 20, 0, 90, 100, 4, 8, HEAD, TAIL, 2, 0

Switch track:
x, y, direction, length, head, tail, divergent, headto(Depreciated), tailto(Depreciated), divergentto(Depreciated), blockid
ex.
swi, 20, 20, 90, 1, 2, 3, TAIL, HEAD, TAIL, 0

Station track:
sta, x, y, rotation, length, head, tail, headto(Depreciated), tailto(Depreciated), id, trackcontrollerid(Depreciated), station_name
ex.
sta, 20, 20, 0, 200, 3,5 TAIL, TAIL, 3, 1, Kingdom Come

Yard track:
yrd, x, y, head, blockid
ex.
yrd, 600, 150, 77, 0

Modification of schedule

To modify the schedule, open red.txt or green.txt, depending on the default schedule you want to change. Station names are seperated from their time (in miliseconds) that the train should target arival. Be sure to spell station names correctly (capitalization matters).
